❓ Frequently Asked Questions

Quick answers to common queries.

Is PDFForge really free? ▼
Yes! PDFForge is 100% free forever. No hidden charges, no watermarks, no registration required.
Are my files safe? ▼
Absolutely. All processing happens locally in your browser. Your files are never uploaded to any server.
Why are some tools marked "Simulated"? ▼
Tools like Word/Excel to PDF require server-side software. We provide demo output so you're not left empty-handed, while our 24 core tools work perfectly.
Do I need to install anything? ▼
No. Just open the website in any modern browser (Chrome, Edge, Firefox).
Is there a file size limit? ▼
There is no hard limit, but very large files (>100MB) may take longer due to browser memory constraints.
